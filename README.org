* 安装
** 下载配置文件
#+BEGIN_SRC shell
git clone https://github.com/wbeiluo/beiluo-emacs.git .emacs.d
#+END_SRC
** 安装字体
#+BEGIN_SRC shell
sudo pacman -S ttf-jetbrains-mono adobe-source-code-pro-fonts wqy-microhei 
#+END_SRC
** 安装其他软件
*** ripgrep
#+BEGIN_SRC shell
sudo pacman -S ripgrep
#+END_SRC
*** SBCL
#+BEGIN_SRC shell
sudo pacman -S sbcl
#+END_SRC
